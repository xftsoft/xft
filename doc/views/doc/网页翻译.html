# 英中翻译

  • Translate English pages into Chinese pages. The software uses Baidu translation interface. Baidu translation has free translation characters every month. If the number of characters exceeds, you need to pay.
  • To open Baidu translation, first register an account with Baidu translation, and then obtain the key of Baidu translation interface.

WARNING

The opening service version of the setting software must be consistent with the opening version of Baidu translation, otherwise there will be many failures limited by Baidu's request. Baidu translation key cannot be used under multiple IP addresses, otherwise it will be blocked by Baidu translation and cannot be used on the same day. If you can't register multiple Baidu accounts, baidu translation will check it and be blocked. You can't use it on the same day.
