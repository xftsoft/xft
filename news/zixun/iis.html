What is an Internet information server (IIS) server?

What is IIS?

IIS is the abbreviation of the full English name of Internet information services. It is a world wide web server service. IIS is a web (web page) service component, including web server, FTP server, NNTP server and SMTP server. It is used for web browsing, file transmission, news service and mail sending respectively. It makes it easy to distribute information on the network (including Internet and LAN).

IIS is the Internet Information Service (Internet Information Service) in the whole process. Its function is to provide information services, such as setting up HTTP and FTP servers. It comes with the system of Windows NT kernel and does not need to be downloaded.

Web application deployed on IIS server

IIS means that you can publish web pages, and ASP (active server pages), Java and VBScript generate pages, with some extended functions. To put it bluntly, IIS belongs to the establishment of windows server. At present, many web servers still install windows server system, such as the most common windows Server2003. The most core function is IIS.

For Xiaobai friends, as long as you know that IIS is a web service component and is used to carry the platform for website running programs, for example, IIS combined with PHP + ASP environment can place the website programs of the current popular ASP / phpx programs. Windows XP / Server2003 comes with IIS6 version, win7 / win8 server version and IIS7 / 8 version. The higher the version, the better the security.

What's the use of IIS?

About the role of IIS and some friends who use it, most of them are website technicians or webmasters. For ordinary computer users, we can see that there is no IIS function in the commonly installed XP / win7 / win8 system, mainly because IIS is useless to ordinary users.
