[software] what are the main functions of flying rabbit site management tool?

function

  • One click Install and uninstall IIS: one click Install, and automatically activate IIS and unlock IIS nodes.

  • Batch site management: manage sites by grouping / labeling / deleting / searching.

  • Quick site creation: right click any folder to create a site and open the index page, and automatically configure permissions / mimeType. If you often edit and debug different website pages, it is very convenient to create and open the page with one click.

  • IIS site backup: backup and restore IIS site data. It can also be used for IIS site migration between different servers.
