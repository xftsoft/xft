Software introduction

Little flying rabbit download has three functional modules. 1. Whole station download module; 2. Page batch download; 3. Batch download of the whole station; 4. Whole station picture download.

  • Whole station download: download the whole website to the local hard disk and browse locally.
  • Web page batch download: download template pages, automatically identify different website codes and convert them into consistent codes.
  • Whole station batch download: it is an extended function of whole station download, giving the whole station download the function of mass download.
  • Whole station picture download: download the pictures of the whole website to the local hard disk.

characteristic

1. Customize the downloaded file structure. 2. Automatically convert the downloaded website code. 3. You can download a large number of pages. 4. Multithreaded speed download. 5. Set custom hyperlink labels, custom picture labels, and custom script labels.
