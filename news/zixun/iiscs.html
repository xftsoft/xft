IIS application pool configuration timeout setting IIS7 configuration timeout

The steps are as follows:

1. First, we need to open the start menu in the lower left corner of the desktop, then open the "management tools" in the "control panel" and open the "Internet Information Service (IIS) manager";

2. After opening, we find the server node on the left of the IIS service manager interface, expand it, and then click "application pool";

3. In the "application pool" page that appears, we select the application pool that needs to configure the limit timeout setting;

4. After selecting, we click "advanced settings" in the operation;

5. In the pop-up dialog box, in "limit timeout (minutes)", enter the number of minutes according to our own needs, and click "OK" after completion.

After completing all the above settings, we have completed the limit timeout setting. The operation is very simple. However, we should remind you to check whether IIS is installed on your computer before configuration. If it is not installed, please install it first and then set it.
